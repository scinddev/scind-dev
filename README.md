# Scind

A workspace orchestration system for Docker Compose.

Site for [scind.dev](https://scind.dev). Built from [`procrastivity/astro-bones`](https://github.com/procrastivity/astro-bones).

## Local dev

    nvm use
    npm install
    npm run dev

## Deploy

Pushes to `main` auto-deploy via Cloudflare Pages.
